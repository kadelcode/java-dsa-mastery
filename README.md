# Java DSA Mastery

A structured repository for mastering:

- Java Programming
- Object-Oriented Programming
- Data Structures
- Algorithms
- Problem Solving
- Software Engineering Fundamentals

## Prerequisites

- Java 21
- Maven 3.9+

## Build

```bash
mvn clean compile
```

## Run Tests

```bash
mvn test
```

## Verify Project

```bash
mvn clean verify
```

## Repository Goals

This repository follows a milestone-driven learning path from Java fundamentals to advanced Data Structures and Algorithms.
